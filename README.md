# Mintlify Clone

This repository is a static clone of the Mintlify landing page / documentation UI built using pure HTML and CSS.  
The goal of this project is to practice real-world UI layout, responsive design, and clean, semantic structure without using any frameworks or JavaScript.

## Preview

This clone recreates the main layout and sections from the original Mintlify site, a modern documentation and landing experience.  [oai_citation:0‡Mintlify](https://www.mintlify.com/?utm_source=chatgpt.com)

## Tech Stack

- HTML5  
- CSS3 (Flexbox + Grid)  
- No frameworks  
- No JavaScript

## How to Run Locally

1. Clone the repository
   git clone https://github.com/prathamd123/mintlify-clone.git

2. Navigate into the folder
   cd mintlify-clone

3. Open the `index.html` file in your browser  
(No server is required; works locally.)
